Mixed Integer Goal Programming for Personalized Meal Optimization with User-Defined Serving Granularity

Mixed Integer Goal Programming (MIGP) was proposed for personalized meal optimization, addressing limitations of existing formulations. MIGP uses integer variables for serving counts and goal programming deviations with inverse-target normalization to balance multi-nutrient optimization. Invisible Orchestrators Suppress Protective Behavior and Dissociate Power-Holders: Safety Risks in Multi-Agent LLM Systems

Multi-agent orchestration systems, where a hidden coordinator manages specialized worker agents, pose significant safety risks due to the orchestrator's invisibility. This architecture has been adopted for enterprise AI deployment without empirical testing of its safety implications. Research findings indicate that invisible orchestration leads to elevated collective dissociation and increased behavioral heterogeneity among workers, while also showing maximal dissociation in itself. PREPING: Building Agent Memory without Tasks

Agents face a cold-start gap when introduced to a new environment without task-specific experience. Prepping, a framework, is proposed to build procedural memory before observing tasks using self-generated synthetic practice. The framework consists of proposer, solver, and validator components that control future practice, execute synthetic tasks, and determine eligible trajectories for memory insertion, providing feedback to guide proposals. Experiments show that Preping improves over a no-memory baseline and achieves performance competitive with strong playbook-based methods.

Conditional Attribute Estimation with Autoregressive Sequence Models

Conditional Attribute Transformers enable three critical capabilities: per-token credit assignment, counterfactual analysis, and steerable generation within a single forward pass. This framework jointly estimates next-token probability and attribute value conditional on each potential next token selection. Sheaf-Theoretic Transport and Obstruction for Detecting Scientific Theory Shift in AI Agents

A finite sheaf-theoretic framework detects scientific theory shift in AI agents by evaluating obstruction measures such as residual fit, overlap incompatibility, constraint violation, limiting-relation failure, and representational cost. The framework tests for gluing source, overlap, target, and validation charts, fitting them to determine the lowest-obstruction candidate for intended deformation or extension.
