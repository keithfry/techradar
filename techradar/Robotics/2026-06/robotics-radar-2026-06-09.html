ROS Discourse General: Context & Hardware Setup

Development of an Autonomous Mobile Robot (AMR) for a muddy dairy barn faces significant localization challenges due to wheel slippage and environmental interference. The current system fuses wheel odometry, IMU data, and UWB ranging using a Dual EKF setup for state estimation ($\text{map} \to \text{odom} \to \text{base\_footprint}$). Major issues include rapid heading (yaw) drift likely caused by magnetic disturbances from steel pillars, and notable positional errors ($10\text{ cm}$ to $40\text{ cm}$) from UWB due to multipath or Non-Line-of-Sight (NLOS) conditions near metal structures. Community advice sought involves strategies to handle magnetometer interference in the IMU and robust outlier rejection methods for corrupted UWB measurements within the EKF framework, potentially exploring UKF or factor graph optimization.

VoLo: A Physical Orchestrator for Open-Vocabulary Long-Horizon Manipulation

Developing robots for open-vocabulary long-horizon manipulation demands reasoning over flexible instructions in complex physical scenes. A key challenge is that the real world does not pause while agents reason, necessitating immediate decision-making. VoLoAgent addresses this by establishing a closed agent loop where a Vision-Language Model (VLM) orchestrates heterogeneous robot tools. It treats large action/world models (VLA/WAM) as interruptible tools it steers during execution alongside vision models and action primitives. The study introduces RoboVoLo, a new benchmark testing long-horizon tasks across common sense and state tracking in real-robot settings.

MinNav: Minimalist Navigation Using Optical Flow For Active Tiny Aerial Robots

MinNav is a novel navigation stack designed for tiny aerial robots using only a monocular camera. It leverages optical flow and its associated uncertainty to enable flight through environments containing static/dynamic obstacles and unknown gaps, requiring no prior scene knowledge. The system enhances success rates by employing an active exploratory movement strategy. Real-world testing demonstrated an overall success rate of 70% across various complex environments. MinNav achieves performance comparable to depth-based methods while requiring significantly less computation, making it suitable for onboard integration on tiny robots.

End-to-End Control of a Powered Knee-Ankle Prosthesis Towards Unified, Tuning-Free Assistance

Introduces an end-to-end prosthesis controller using Temporal Convolutional Networks to estimate actuator signals directly from onboard sensors. This approach eliminates the need for explicit mode classification and complex manual tuning in powered prostheses. The system was trained on a multi-terrain dataset involving 18 transfemoral amputees and was tested across five locomotion modes. During testing, the controller demonstrated accurate scaling of ankle torque with walking speed during level ground ambulation. Furthermore, it successfully generated seamless stair transitions and appropriately adjusted resistive knee torque for ramp descent without subject-specific tuning.

X-OP: Cross-Morphology Whole-Body Teleoperation via MPC Retargeting

A new hierarchical framework enables cross-morphology whole-body teleoperation using only a single Extended Reality (XR) device, avoiding costly hardware setups. It employs a Model Predictive Control (MPC)-based motion retargeter to balance the operator's intent with the robot's dynamic feasibility. The system incorporates state synchronization and SLAM feedback for robust online execution against noise and drift. Simulation results demonstrate superior performance on whole-body tasks for both humanoid and mobile manipulator platforms compared to prior methods. This plug-and-play approach offers a scalable, morphology-agnostic solution allowing real-time behavioral customization across various robots.

IntentNav: Learning Spatial-Visual Object Navigation from Human Demonstrations

Object navigation tasks require a robot to explore an unknown environment for an unobserved target using human-like search strategies. IntentNav is presented as a spatial-visual imitation framework that learns these policies directly from human demonstrations. It introduces Frontier-based Human-Intent Labeling to infer high-level search intent by labeling the most plausible frontier based on future demonstrator actions. The system maintains a candidate space tracking explored regions, unexplored frontiers, and trajectory history alongside semantic visual cues. IntentNav uses a Vision-Language Model (VLM) policy trained with an Intent-Aligned Objective, achieving state-of-the-art results across multiple benchmarks and enabling zero-shot transfer to various robot types.

MuJoCo-Drones-Gym: A GPU-Accelerated Multi-Drone Simulator for Control and Reinforcement Learning

MuJoCo-Drones-Gym is an open-source, Gymnasium-compatible multi-drone simulator built using the MuJoCo physics engine for aerial robotics research. It supports arbitrary numbers of Bitcraze Crazyflie 2.x quadcopters with a modular API allowing selection of physics models, action interfaces, and observation spaces. The environment integrates seamlessly into multi-agent reinforcement learning via a PettingZoo ParallelEnv wrapper. It includes seven distinct task environments, such as waypoint navigation and formation flight, demonstrating its versatility. Key advantages include leveraging MuJoCo's advanced features like improved contact handling and parallelizability for RL training.

EgoAERO: Learning Dexterous Manipulation from a Single Egocentric Video without Object Assets

EgoAERO is a novel framework designed to learn dexterous manipulation directly from single egocentric RGB-D videos, overcoming the limitations of missing object assets in existing data. It reconstructs contact-consistent hand-object trajectories by employing asset-free object tracking and reconstruction, alongside ego motion compensation. The framework then converts these learned trajectories into robot policies using a two-stage residual learning approach. Furthermore, it introduces an online quality assessment mechanism and creates EgoDex-R, a large dataset containing 4.3 million RGB-D frames for policy training. Experimental results demonstrate that EgoAERO enables single-demonstration dexterous manipulation with performance comparable to CAD-based reconstructions on the HOI4D task.
