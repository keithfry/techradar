ROS Discourse General: A VS Code extension for quickly running ROS2 launch files, Python nodes, and C++ nodes

Similarly, right-clicking Python or C++ nodes triggers `ros2 run`, handling workspace sourcing automatically. The extension also provides functions to source the workspace and build the entire ROS 2 workspace using `colcon build`. It intelligently detects the ROS 2 workspace by searching for directories containing package definitions.

Researchers used it for sustained experiments demonstrating that RL policies can learn directly on robots. Findings highlight the critical need for on-device adaptation due to performance degradation from minor distribution shifts between training and deployment.

DiffusionVS: A Generative Framework for Robust Visual Servoing Based on Diffusion Policy

Introduces DiffusionVS, a novel diffusion-based framework for improving robust visual servoing in robotics. The method utilizes normalized image coordinates of observed tag corners to generate camera velocity via conditional denoising based on Diffusion Policy. To improve generalization beyond static datasets, it employs an online training paradigm that continuously expands training data through interactive experience collection. Simulations show near 100% success rates, and real-world experiments achieved a successful rate of 93%. Furthermore, integrating the diffusion mechanism demonstrably enhances performance across various existing visual servoing networks.

Playful Agentic Robot Learning

Current robot systems typically acquire reusable skills only through explicit, task-driven instructions. Playful Agentic Robot Learning introduces an approach where embodied coding agents utilize self-directed play for continual skill acquisition. Researchers developed Robotics Agent Teams (RATs) which propose exploratory tasks, execute policies, and diagnose failures during play. Experiments demonstrated that skills learned through this "play" stage significantly boost performance on held-out tasks compared to non-play baselines.

Simulating Robotic Locomotion in Sand: Resistive Force Theory in an Open-Source Physics Engine

Novel applications of Resistive Force Theory (RFT) allow for approximating ground reaction forces when simulating robot movement in sand without detailed grain-by-grain modeling. Integrating 3D Granular RFT into the physics engine MuJoCo enables simulations for freely walking robots on such substrates. The developed tool successfully predicts key behaviors, such as foot sinkage and walking distance, with an accuracy of within 20\% compared to experimental results. Testing across various scenarios confirmed that crucial trends related to end effector shape, speed, and loading remain accurate. This open-source implementation shows promise for advancing the design of robots intended to traverse granular media.

Proprioceptive Invariant State Estimation for Humanoid Robots on Non-Inertial Ground

Introduces an invariant extended Kalman filtering (InEKF) method for estimating humanoid robot states on moving ground using only internal proprioceptive sensors. This approach estimates the base position and velocity relative to the non-inertial ground frame, eliminating the need for external motion measurements. It utilizes kinematic constraints at the stance foot via foot-mounted IMUs to handle nonlinearities from ground motion. The resulting estimator has a right-invariant measurement model, beneficial when initial uncertainties are large. Experiments on swaying/pitching ground showed a 96% faster convergence and an 80% error reduction compared to prior InEKFs.

A Categorial and Sheaf-Theoretic Semantics for Autonomic Component Ensembles

Large-scale autonomous agent systems challenge traditional formal methods. The Software Component Ensemble Language (SCEL) requires an improved operational semantics for global property reasoning. This report introduces a multi-layered model using category theory and sheaf theory to analyze SCEL. It posits that robot ensembles can be modeled as a sheaf over a topological space, linking components to points and knowledge to data. Consequently, processes like information sharing become equivalent to "gluing" local data, allowing system failures to be quantified as measurable topological obstructions.

SCAN-Planner: Spatial Collision-Aware Local Planning for Route-Guided Long-Range Quadruped Navigation

SCAN-Planner is a novel, spatial collision-aware local planning framework designed for quadruped robots navigating complex environments. It addresses limitations of existing methods by using a yaw-aware twin-cylinder footprint to model the robot's elongated body for whole-body collision checks in 3D occupancy maps. The planner generates collision-free guidance using a projected A* search on an interpolated ground surface, which also manages vertical stability. For large-scale tasks, it incorporates a robot-centric sliding map that offers high-resolution local collision checking and recovery capabilities. Testing shows SCAN-Planner successfully produces safe, smooth, and efficient trajectories in dense clutter, unstructured 3D scenes, and stair traversal scenarios.

pdSTL: Probabilistic Differentiable Signal Temporal Logic for Stochastic Systems

Developed pdSTL (probabilistic differentiable Signal Temporal Logic) to handle complex temporal specifications for robots in uncertain environments. This framework combines probabilistic semantics with differentiability over belief trajectories, addressing limitations of existing methods. It uses interval-valued probabilistic semantics to compute conservative satisfaction bounds within the STL syntax tree. The temporal robustness evaluation is formulated as a recurrent, LSTM-style unfolding, allowing efficient, differentiable monitoring. Validation on obstacle avoidance and quadcopter flight confirms pdSTL achieves optimization with formal probabilistic guarantees superior to deterministic approaches.
