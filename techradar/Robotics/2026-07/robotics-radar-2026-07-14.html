ROS Discourse General: LinkForge: Exploring an Intermediate Representation (IR) for Robot Descriptions

LinkForge evolved from a Blender extension to address challenges in robot description authoring formats like URDF and XACRO. The project introduces a programmable Intermediate Representation (IR) designed to improve this process. This IR enables programmatic composition of robot models and validates kinematic/physical properties before export. It aims to function as a tool-agnostic layer preceding existing ROS workflows, rather than replacing them entirely. Authors are seeking community feedback on the need for such an intermediate representation in current robot description pipelines.

ROS Discourse General: RoboShield: A Low-Latency Out-of-Band RTPS Watchdog in Rust

RoboShield is a low-latency, out-of-band real-time RTPS security watchdog written in Rust. It intercepts and parses raw DDS/RTPS packets from UDP captures to detect anomalies on the network. The system validates each packet's GUID against a configurable whitelist and monitors topic message rates for flood attacks. Unlike solutions requiring node modification like SROS2, RoboShield plugs directly into the network, imposing no changes on existing nodes. Benchmarks show an average processing time of approximately 1.36 μs per packet on x86 hardware.

EgoSteer: A Full-Stack System Towards Steerable Dexterous Manipulation from Egocentric Videos

EgoSteer is a full-stack system designed to enable steerable dexterous manipulation using egocentric human videos. It tackles the lack of suitable demonstration data by integrating EgoSmith, a new pipeline generating 9.6K hours of high-quality pre-training material from in-the-wild videos. The architecture features a unified robot stack for teleoperation and human corrections. After language-guided pre-training on this data, the system undergoes robot post-training and DAgger refinement. Evaluations show EgoSteer robustly executes over 40 diverse tasks, demonstrating generalization and few-shot adaptation to complex goals like box folding.

RoboNav-Arm: Agentic AI-Driven Navigation and Obstacle Avoidance for Robotic Manipulator in Cluttered Environments

Addresses safety challenges for robotic manipulators in unstructured environments facing unforeseen obstacles. A novel framework incorporates an environment module for real-time obstacle detection, 3D localization, and geometry estimation, producing a semantic report. Central coordination manages system communication and updates tools like memory and collision scenes. The planning module dynamically selects the best motion planner (e.g., RRTConnect) based on the current goal. This approach refines generated trajectories to guarantee safe, collision-free task execution, demonstrated robustly in Gazebo Classic.

arXiv - cs.RO

Learning High-Level Decision Making with an Interaction-Aware Attention-Based Network in Autonomous Driving

Developing high-level decision making for autonomous driving requires handling dynamically sized inputs due to varying traffic. Existing methods either neglect explicit traffic interactions (like DeepSet) or suffer from high computational complexity when modeling them using attention mechanisms. DecisionPerceiver is proposed, adapting the Perceiver IO architecture by projecting dynamic agent features into a fixed-size latent space via tunable latent queries. This approach improves scalability for handling larger networks while retaining interaction awareness. Evaluation across multiple driving scenarios confirms consistent performance gains and successful generalization even as vehicle density increases.

arXiv - cs.RO

SWIFT: A Small-World Interaction Framework for Flow-Aware Trajectory Prediction in Autonomous Driving

Accurate trajectory prediction for autonomous driving requires modeling complex, context-dependent agent interactions beyond purely data-driven methods. The paper proposes SWIFT (Small-World Interaction Framework), a unified model that merges small-world networks with traffic flow theory to incorporate structural priors. SWIFT employs a Small-World Interaction Network to capture both local and global dependencies, alongside a Flow Regime Encoder for scene adaptation. Additionally, a multi-relational graph module enhances interaction reasoning by explicitly encoding agent relationships. Experiments on nuScenes, MoCAD, and NGSIM demonstrate that SWIFT surpasses strong baselines in accuracy, generalization, and robustness across varied traffic conditions.

arXiv - cs.RO

OmniSCS: Omni Safety-Critical Scenario Synthesis for Autonomous Driving via a Fully Editable Driving World

Developing robust autonomous driving systems requires synthesizing and simulating safety-critical scenarios (SCS). Existing methods struggle with maintaining data fidelity when editing agent states or scene elements in existing environments. The proposed OmniSCS system addresses this by generating photorealistic, high-physical-fidelity SCS within synthetic settings for closed-loop testing. It features a module that reconstructs agents and backgrounds while preserving appearance during editing, alongside an SCS Synthesis module for object insertion and trajectory manipulation. Testing on nuScenes, Waymo, and KITTI datasets confirms OmniSCS surpasses current methods in edited scene fidelity and supports real-time validation of self-driving algorithms.

arXiv - cs.RO

A Risk-Field Enhanced Closed-Loop Digital Twin Framework for Autonomous Driving Safety Validation

Developing reliable safety validation for autonomous driving necessitates overcoming the limitations of expensive real-world testing. This paper proposes a risk-field enhanced closed-loop digital twin framework to address this gap. The framework integrates physical data acquisition with virtual reconstruction, scenario generation, and algorithm evaluation. A novel "driving risk field" is introduced as a unified representation quantifying various risks around the ego vehicle. Implementing this risk structure allows for targeted, interpretable safety validation, though performance relies on model fidelity and sim-to-real transfer accuracy.

arXiv - cs.RO

A Comprehensive Survey and Systematic Real-World Evaluation of Embodied Vision-and-Language Navigation

Navigation is crucial for autonomous systems, but current methods often lack real-world robustness. Vision-and-Language Navigation (VLN) integrates language understanding with visual perception for data-driven navigation solutions. This survey systematically reviews VLN by classifying state-of-the-art methods across action and model paradigms. Furthermore, the authors conducted a real-world evaluation on ten scenes, revealing a significant performance drop from simulation to physical deployment. Specifically, while one RGB-only method failed substantially in reality (61% to 22%), a hierarchical framework showed better robustness (51% success rate).
