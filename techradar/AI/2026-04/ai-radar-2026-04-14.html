AHC: Meta-Learned Adaptive Compression for Continual Object Detection on Memory-Constrained Microcontrollers

Adaptive Hierarchical Compression (AHC) is a meta-learning framework designed for continual object detection on memory-constrained microcontrollers (MCUs). Experiments show AHC achieves competitive accuracy on multiple benchmarks while enabling practical continual detection within the tight memory constraints.
