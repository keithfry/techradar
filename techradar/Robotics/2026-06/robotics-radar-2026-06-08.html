ROS Discourse General: Pre-built ROS2 Humble Windows SDK + Standalone RViz2 (No Toolchain Required)

Addresses the difficulty of setting up ROS2 on Windows by providing two key tools. Ros2Simple offers a pre-built ROS2 Humble SDK for Windows x64, allowing development using standard CMake without a native ROS2 toolchain or `colcon`. Rviz2Windows provides a standalone, zero-configuration RViz2 executable built from ROS2 Humble source, requiring no prior ROS2 installation. These resources simplify workflows for industrial deployment, education, and rapid prototyping on Windows platforms. The solutions are compatible with other DDS-enabled ROS2 nodes.

ROS Discourse General: Isaac Lab Teleoperation & Data Collection: Controlling the AgileX NERO 7-DoF Robotic Arm

The tutorial details implementing keyboard teleoperation and data collection for the AgileX NERO (7-DoF) robotic arm within NVIDIA Isaac Lab. Users can achieve full control of the simulated arm via direct keyboard input for SE(3) pose alignment. Functionality includes simulating a multi-block stacking (Pick and Place) task, logging human demonstration data to HDF5 format compatible with Robomimic, and replaying these collected trajectories for imitation learning verification. The setup supports both NERO and PiPER robotic arms out of the box. Preparation requires merging multiple XACRO files into a unified URDF file for Omniverse integration.
