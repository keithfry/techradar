MixAtlas: Uncertainty-aware Data Mixture Optimization for Multimodal LLM Midtraining

MixAtlas addresses the limited exploration of data mixture optimization during multimodal LLM midtraining by decomposing the training corpus across two axes: image concepts (10 clusters from CLIP) and task supervision (5 objectives). The method uses small proxy models with a Gaussian-process surrogate and GP-UCB acquisition to efficiently search this joint mixture space. During evaluation on ten benchmarks, optimized mixtures significantly boosted average performance on Qwen2-7B by 8.5%-17.6% compared to existing methods. Furthermore, these optimized mixtures enabled reaching baseline training loss in up to half the necessary steps. A key feature is the transferability of recipes discovered on smaller proxies to larger, 7B-scale models.

MinShap: A Modified Shapley Value Approach for Feature Selection

Feature selection remains a difficult problem, particularly with unknown non-linear and dependent feature relationships. Existing Shapley value methods are ill-suited for selection because they capture both direct and indirect feature effects. This work proposes MinShap, a modification that replaces the average marginal contribution with the minimum marginal contribution across feature permutations. Theoretically, MinShap is grounded in the faithfulness assumption from Directed Acyclic Graphical Models (DAGs) and offers guarantees for Type I error. Simulations demonstrate that MinShap outperforms state-of-the-art methods like LOCO, GCM, and Lasso in accuracy and stability.

Structural interpretability in SVMs with truncated orthogonal polynomial kernels

Develops Orthogonal Representation Contribution Analysis (ORCA) to enhance post-training interpretability of Support Vector Machines (SVMs) using truncated orthogonal polynomial kernels. Because the associated reproducing kernel Hilbert space is finite-dimensional with a known basis, the decision function can be exactly expressed in intrinsic RKHS coordinates. The Devil Is in Gradient Entanglement: Energy-Aware Gradient Coordinator for Robust Generalized Category Discovery

Generalized Category Discovery (GCD) methods suffer from gradient entanglement, which degrades known class discrimination and causes overlap between known and novel class representations. To solve this, the Energy-Aware Gradient Coordinator (EAGC) proposes a gradient-level module for optimization regulation. EAGC contains two parts: Anchor-based Gradient Alignment (AGA) and Energy-aware Elastic Projection (EEP). AGA anchors labeled sample gradients using a reference model, maintaining known class structure. EEP softly projects unlabeled gradients away from the known-class subspace using an energy-based coefficient for adaptive scaling. Experiments confirm that EAGC enhances existing techniques and achieves new state-of-the-art performance.
