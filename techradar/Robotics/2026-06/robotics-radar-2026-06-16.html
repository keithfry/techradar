Pretrained to Imagine, Fine-Tuned to Act: The Rise of World-Action Models

World-Action Models (WAMs) represent an advancement in robot policy design. These models begin with a pre-trained Vision-Language Model (VLM) backbone for initial capabilities. They are then fine-tuned to translate high-level linguistic instructions into physical actions. Essentially, WAMs bridge the gap between understanding natural language and executing complex physical tasks. This architecture allows robots to move beyond simple recognition toward goal-directed action planning.

Bayesian Optimization for Learning Nonlinear MPC in Autonomous Agent Navigation

A map-free framework integrating reactive rolling-horizon planning with nonlinear Model Predictive Control (MPC) addresses autonomous navigation in unknown settings. The system uses LiDAR data to create Gaussian occupancy maps, generating collision-free trajectories via A* search tracked by an MPC formulation. Robustness against parameter sensitivity is enhanced using an offline Bayesian optimization scheme based on Tree-structured Parzen Estimators (TPE). A Gaussian Process surrogate aids in analyzing the resulting parameter landscape. Evaluation on a Unitree Go2 quadruped demonstrated that simulation-tuned parameters maintained high performance on hardware, achieving up to 90.0% navigation success.

Synthetic-to-Real Pipeline for Safe Landing Zone Detection

Autonomous Uncrewed Aerial Vehicle (UAV) safe landings in unstructured areas require precise semantic understanding despite limited annotated aerial data. The proposed solution introduces a perception and data generation pipeline tackling the sim-to-real gap. This system uses a procedural synthetic engine to create photorealistic, automatically annotated urban environments through domain randomization. A OneFormer Transformer is fine-tuned on this generated data for global context resolution in segmentation. Finally, a deterministic module employs an Euclidean Distance Transform to identify safe landing zones with guaranteed obstacle clearance, proving effectiveness via UAVid benchmarking and real-world validation.

Deep Learning-Based Lunar Crater Terrain Relative Navigation

Accurate position estimation for autonomous lunar landings requires robust Terrain Relative Navigation (TRN). A proposed algorithm integrates a deep-learning crater detector with an Extended Kalman Filter (EKF). The detector analyzes monocular images from orbit, matching observed craters against a global database using Hungarian assignment and outlier removal. These identified measurements refine the EKF, estimating spacecraft pose in the LCLF frame while constraining radial drift with altitude aiding. Simulations show the TRN can recover navigation accuracy to hundreds of meters even after an initial positional error of up to 5 km.

TacStyle: Personalizing Tactile Robot Policies using Structured Behavior Representations

To allow assistive robots to adapt to individual user tastes, the paper proposes using language to reason about preferred behaviors rather than directly generating them. The method learns a structured latent representation that organizes these user preferences based on trajectory differences. When given a preference prompt, a foundation model interprets this latent space to select a specific value corresponding to the desired behavior. Experiments demonstrate that selecting robot behaviors from this structured latent space enables more precise adaptation to user tastes. This approach achieves better results while requiring substantially fewer preference labels than direct language-conditioned policies.

arXiv - cs.RO

VANDERER: Map-Free Exploration using Future-Aware and Visual-Curiosity-Guided Diffusion Policy

VANDERER is an exploration framework designed for mobile agents to navigate unseen environments without relying on accurate occupancy maps, which are difficult to generate with limited sensors like monocular cameras. It guides pre-trained diffusion policies by incorporating a Visual Curiosity Module (VCM). This VCM utilizes a navigation world model to predict the outcomes of potential actions and quantifies them using a curiosity cost. The resulting cost actively steers the diffusion process toward generating actions that maximize exploration within the environment. Empirical testing across diverse simulated settings showed VANDERER significantly surpassed existing methods, exploring an average of 13.4% more area than NoMaD.

arXiv - cs.RO

Multimodal Physiological Assessment of Contact-Rich Physical Human-Robot Interaction Under Varying Environmental Conditions

Investigated physiological responses during contact-rich physical human-robot interaction across 18 environmental combinations varying temperature, noise, and light. Multimodal data—including EDA, sEMG, eye-tracking, and subjective ratings—were collected alongside task performance metrics. Results indicated that while objective performance remained constant regardless of environmental stress, autonomic workload (SCL) increased with elevated temperatures. This suggests operators compensate for thermal discomfort by increasing physiological effort to maintain consistent execution. These findings recommend developing control systems that incorporate real-time physiological data to actively reduce operator workload in complex environments.

arXiv - cs.RO

Inference-time Policy Steering via Vision and Touch

ViTaL is a framework that steers pre-trained generative robot policies during deployment by using both visual and tactile information for action verification. It models multimodal guidance as a bi-level optimization problem: high-level visual sampling selects the overall behavior, while low-level tactile diffusion editing refines the action sequence to meet local contact requirements. To support outcome-based steering, ViTaL learns a visuo-tactile latent world model and utilizes semantically aligned verifiers, including a text-conditioned tactile reward in latent space. Testing across three real-world, contact-rich tasks showed that ViTaL increased overall success by 51% compared to the base policy. The framework notably outperformed unimodal steering approaches by at least 33%.

LV-Calib: LiDAR-Camera Extrinsic Calibration with Boundary-Response Modeling

LV-Calib is a new framework for estimating LiDAR-camera extrinsic parameters and performing LiDAR boundary-response calibration using a printable planar target. The target simultaneously provides indexed visual measurements via fiducials and structural feature points from circular reflectivity boundaries detectable by LiDAR. Instead of assuming ideal contours, the method refines accurate 3D LiDAR features by handling background point cropping and the broadened transition band at discontinuities. This leads to a weighted reprojection-consistent extrinsic optimization using image observations and refined LiDAR feature alignment. Finally, LV-Calib calibrates the LiDAR boundary response by estimating pitch-yaw-range residual statistics from overlapping boundary samples, achieving high accuracy in experiments.
