The Secret to Marathon-Winning Humanoid Robots

Honor's Lightning robot achieved its fast half-marathon time using sophisticated motor gearing optimized for running mechanics. The key enabling technology is its liquid cooling system circulating through independent circuits in all four lower limb motors, which manages significant waste heat. Running demands a specific trade-off between motor efficiency and power management that air cooling cannot sustain over distance. Competitors often struggle because designing for one task (like walking) imposes inefficient hardware when running at speed, or vice versa. Ultimately, the performance highlights careful engineering optimization rather than magical leaps in technology.

ROS Discourse General: Has anyone tried conformal prediction for sensor gating in a nav stack?

The original method for sensor gating often relies on chi-squared gates assuming Gaussian noise. This assumption breaks down in real-world outdoor environments due to factors like multipath effects or tree cover. Conformal prediction offers an alternative that provides coverage guarantees without assuming a specific noise distribution, relying instead on empirical data. The author is inquiring whether others have implemented this technique in navigation stacks. Ultimately, the poster questions if fixing the underlying mathematical assumption is worth the effort compared to simpler tuning methods for GPS covariance mismatch.

Recover, Discover, Plan: Learning Skills and Concepts from Robot Failures

Intelligent robots must learn to avoid failures proactively by acquiring underlying knowledge beyond mere reactive recovery. A new method called Recovery-Driven Synthesis of Relational Concepts (ReSYNC) addresses this limitation efficiently. ReSYNC uses an incremental dual-learning process: first, it learns recovery skills via reinforcement learning. Second, it discovers and refines abstract relational predicates to build a general planning model explaining these recoveries. This cycle allows the system to convert specific training fixes into broad failure avoidance capabilities for unseen problems. Testing across four simulated domains demonstrated ReSYNC's ability to solve complex tasks, significantly outperforming existing methods, with proof-of-concept transfer shown to real-world manipulation.

Guava: An Effective and Universal Harness for Embodied Manipulation

Guava introduces a harness framework designed to facilitate embodied tool use for language models. It suggests that effective embodied agents require three core components: iterative perception-reasoning-action loops, semantic action abstractions, and multimodal observations. The authors trained an end-to-end pipeline distilling these capabilities into a 4B open-source model using under 2K simulated trajectories. Evaluation in both simulation and real environments showed comparable performance to leading proprietary models. Ultimately, Guava proves that a structured harness can serve as a scalable, model-agnostic interface for achieving strong embodied manipulation with minimal data.

PAIWorld: A 3D-Consistent World Foundation Model for Robotic Manipulation

Current world foundation models fail in robotic manipulation because they lack 3D consistency across multiple camera views. These failures stem from insufficient inter-view communication and the absence of a strong 3D geometric prior. PAIWorld addresses these shortcomings by augmenting diffusion-transformer models with three key components. These include Geometry-Aware Cross-View Attention, Geometric Rotary Position Embedding for encoding pose information, and Latent 3D-REPA for maintaining 3D feature consistency. The resulting framework achieves state-of-the-art multi-view 3D consistency on benchmarks like WorldArena and AgiBot-Challenge2026.

arXiv - cs.RO

VEGA: Learning Navigation VLAs from In-the-Wild Egocentric Video with Geometric Trajectory Supervision

VEGA introduces a method to train navigation VisionLanguage-Action (VLA) models using vast amounts of unlabeled egocentric videos. It bridges the gap between raw video data and policy learning by reconstructing local scene geometry from monocular video input. This geometry allows VEGA to generate obstacle-aware trajectories conditioned on explicit navigation goals, which are then used for training a flow-matching VLA policy. The approach distills necessary obstacle-aware planning directly into the vision-based policy using geometric supervision exclusively during training. Evaluation on the new benchmark, VEGA-Bench, shows significant improvements over baseline models in goal progress, collision reduction, and success rates in both simulated and real-world tests.

arXiv - cs.RO

Task Allocation and Motion Planning in Dynamic, Cluttered Environments via CBBA and Graphs of Convex Sets

Addresses multi-agent task planning in complex, changing environments by integrating task assignment with trajectory generation. The proposed solution utilizes Graphs of Convex Sets (GCS) to optimize safe and efficient paths within a time-extended 3D configuration space. Simultaneously, the Consensus-Based Bundle Algorithm (CBBA) handles distributed coordination for task allocation among agents. By connecting these two methods, the system enables informed task assignments while ensuring collision avoidance in the dynamic environment. Simulations confirm the approach's effectiveness for both static and dynamic tasks within cluttered settings.

arXiv - cs.RO

DREAM-Chunk: Reactive Action Chunking with Latent World Model

Action chunking is a common interface for vision-language-action (VLA) models but suffers from brittleness during open-loop execution due to stochastic dynamics or errors. DREAM-Chunk proposes a test-time scaling method that enhances chunking policies by integrating a lightweight latent world model without retraining. At inference time, it samples multiple candidate action chunks and simulates their predicted latent futures. It then selects actions from the chunk whose forecasted state best matches the actual observed rollout. Evaluations on benchmarks and physical robots show DREAM-Chunk significantly improves robustness against various sources of stochasticity during long-horizon task execution.

arXiv - cs.RO

Benchmarking Action Spaces in Reinforcement Learning for Vision-based Robotic Manipulation

Action space selection critically impacts motion smoothness, safety, and performance in real-world reinforcement learning (RL) for robotics. The study benchmarks four action types—pose increment, pose velocity, joint position increment, and joint velocity—across object picking and pushing tasks using sim-to-real transfer. Evaluating these representations demonstrated that the choice of action space significantly affects real-world deployment performance. Specifically, joint velocity proved to be the optimal representation for both vision-based picking and pushing tasks regarding smoothness and final task metrics. Practical guidelines are provided to help RL practitioners select appropriate action spaces for simulation and physical experiments.

arXiv - cs.RO

Admittance-Based Surface Alignment for Human-in-the-Loop Robotic Visual Inspection

Precision visual inspection in industries like aerospace demands precise robotic end-effector alignment despite surface irregularities and noise. Traditional offline motion planning fails when incorporating real-time human adjustments via teleoperation or shared autonomy. This research introduces a novel, real-time, closed-loop control pipeline using an admittance-based framework. It models the end-effector as a virtual sphere moving through a viscous medium to generate compliant motion. Validation on a 6-DOF manipulator successfully demonstrated stable normal-tracking and achieved a mean orientation error of only $0.4^\circ$.
