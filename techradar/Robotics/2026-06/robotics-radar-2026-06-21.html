ROS Discourse General: AIC Phase 1: How to run custom aic_model service in Flowstate with AIC_ROUTER_ADDR?

A user is attempting to run a custom `aic_model` service instance within the Flowstate environment for AIC Phase 1. The required baseline endpoints are `/aic_model/change_state` and `/insert_cable`, using a local Docker image. Key questions involve whether Flowstate automatically provides necessary environment variables like `AIC_ROUTER_ADDR` and handling of secrets such as `AIC_MODEL_PASSWD`. The author also seeks confirmation on the correct sequence, utilizing `inctl asset install` and `inctl service add`, for making their participant model available for testing.

ROS Discourse General: Introducing CycloneDDS Insight - Graphical DDS Inspection and Debugging for ROS 2

CycloneDDS Insight is a graphical tool designed for inspecting and debugging DDS within ROS 2 and Eclipse Cyclone DDS environments. It grants users visibility into DDS participants, topics, publishers, and subscribers to understand system topology. Key features include live inspection of DDS entities, communication debugging, and visualization of the overall communication architecture. The tool also supports monitoring DDS traffic and statistics across complex multi-host deployments. Being free and open source, it makes advanced debugging capabilities accessible throughout the ROS 2 community.

ROS Discourse General: Scale Models of ROS robots

An author constructed a functional 1/5 scale model of the Husarion Lynx using 3D-printed PLA for the shell and off-the-shelf crawler wheels. The project incorporates small microservos and is powered by an expensive CM4 board, overcoming space constraints with creative modifications. Navigation was achieved by simulating LiDAR input using a Vl53L7CX sensor combined with wheel odometry and BNO085 data. Thermal management proved to be a major challenge, necessitating fans and vents due to the high power density in the compact chassis. The detailed setup documentation is shared via GitHub, intended as inspiration for other model builders.
