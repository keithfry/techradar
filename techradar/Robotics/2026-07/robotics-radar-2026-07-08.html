Quaternion-Averaging-Based Adaptive Complementary Filter for Pedestrian Dead Reckoning With a Foot-Mounted AHRS

Pedestrian Dead Reckoning offers an indoor navigation solution unaffected by GPS signal degradation. The accuracy of foot-mounted AHRS-based PDR heavily relies on the employed attitude estimation algorithm. A Quaternion-Averaging-Based Adaptive Complementary Filter (QAACF) is proposed to improve this accuracy while lowering computational demands. QAACF integrates quaternions from angular velocity with those from acceleration and magnetic field using Markley's quaternion averaging. The method further adaptively adjusts sensor weights based on gait phases and magnetic disturbances, showing lower RMSEs than existing filters with reduced computational overhead compared to Kalman filters.

arXiv - cs.RO

Learning 4D Geometric Priors for Inference-Efficient World Action Models

World Action Models (WAMs) for robotics often fail to capture necessary temporal geometry using appearance-based video latents. MECo-WAM introduces a Multi-Expert Co-Training approach by injecting action-relevant 4D geometric priors into video-action representations. During training, the model uses a lightweight 4D expert supervised by relational targets from a frozen encoder and employs asymmetric visibility to prevent spurious shortcuts. A decayed 4D read-mask attention progressively transfers this geometric knowledge while minimizing deployment cost. Evaluation on datasets like LIBERO and RoboTwin confirms that MECo-WAM improves manipulation performance significantly without increasing inference complexity.

arXiv - cs.RO

GEM-Occ: From Visual Geometry Evidence to Embodied Semantic Occupancy Memory

Semantic occupancy memory is crucial for embodied agents navigating connected indoor spaces by structuring representations of occupied regions, free space, unknown areas, and object semantics. Recognizing gaps in current research focusing on single-view or room-level perception, the authors introduce HIOcc, a benchmark unifying multiple indoor datasets into a consistent sparse semantic format supporting local, room-level, and building-level evaluation. They propose GEM-Occ, a Gaussian Evidence Memory framework that treats local visual predictions as transient evidence rather than fixed pointmaps. GEM-Occ converts this evidence into semantic Gaussian occupancy and free-space ray data for persistent storage via causal updates. This hierarchical memory structure—comprising caches, submaps, and a building graph—improves performance across all tested aspects compared to prior baselines on the HIOcc benchmark.

Dynamic Evaluation of Classical and Control-Aware Optimal Trajectory Planning in Robot Manipulators

Trajectory planning significantly impacts robot manipulator performance metrics like tracking accuracy and actuator demand. Classical planners are purely kinematic, leading to inefficiencies when executing nominally smooth paths due to ignored system dynamics. This paper introduces a control-aware optimal trajectory planning framework incorporating explicit manipulator dynamics and actuator effort into a finite-horizon formulation. A unique unified evaluation framework tests all planners under identical closed-loop nonlinear execution conditions. Simulations on an UR5 manipulator demonstrate that the proposed method consistently reduces tracking error, corrective torque, and overall execution cost compared to classical approaches.

Physics-Regularized Machine Learning for Proprioceptive Vehicle Localization Using Onboard Sensors

Accurate vehicle localization is critical for autonomous systems, but performance degrades during sensor outages despite IMU/satellite fusion. This paper introduces Physics-Regularized Machine Learning for Localization (PRML2), a hybrid framework combining Kalman filtering and data-driven learning. PRML2 leverages physics-regularization by training an ML model through a differentiable Kalman filter, ensuring consistency with vehicle motion models. This approach enhances localization accuracy and generalization across various driving conditions. Testing shows PRML2 achieves superior, real-time localization performance using only onboard sensors and introduces a novel dataset for low-friction research.

Efficient Transfer Learning of Robot Dynamic Models Using Morphological Similarity

A new neural network framework proposes transfer learning to model soft, fin-actuated underwater robot dynamics. This approach specializes in handling morphologically similar robots that vary only in scale and hydrodynamic traits. The method utilizes an autoencoder-based domain adaptation technique to create a shared latent representation of the robot dynamics. By aligning this representation, it facilitates efficient knowledge transfer from a larger source robot to a smaller target robot. Testing on two real underwater robots confirmed accurate state estimation on the target platform without requiring labeled data for fine-tuning.

Uncertainty-Aware Velocity Correction for Proprioceptive Vehicle Localization using Evidential Mamba

Addressing reliable localization challenges in GPS-denied environments, the authors propose Evidential Velocity Correction using Mamba (EVC-Mamba). This novel, learning-based architecture estimates a virtual velocity sensor directly from onboard vehicle data, eliminating the need for extra hardware. It combines a Mamba selective state space model to capture motion dynamics with evidential deep learning for principled uncertainty quantification. The resultant uncertainty-aware velocity estimate corrects IMU drift when fed into an Error-State Extended Kalman Filter. Testing on real data showed this method achieved localization accuracy within 10% of dedicated external sensors, supporting real-time operation on edge hardware during extended GPS outages.

IMR: Iterative Mode-World Weighted Regression for Multi-Agent Trajectory Prediction

Multi-agent motion prediction is crucial for safe automated vehicle operation, as previous methods have limitations in mode diversity or prediction accuracy. To overcome these shortcomings, a mode-world weighted regression loss is introduced to improve both world ranking and top-1 confidence while mitigating mode collapse. The proposed framework also incorporates an iterative decoder to enhance prediction accuracy through recurrent and segmental trajectory generation. Empirical testing demonstrates that this novel approach achieves the highest ranking on the Argoverse 2 multi-agent motion forecasting benchmark.

Observation Quality Matters: Robust Multi-Fisheye Calibration via Failure-Oriented Analysis

Calibrating multi-fisheye camera systems is challenging due to increasing complexity and reliance on high-quality observations. Research reveals that calibration failures are primarily caused by intrinsic initialization issues when limited radial span data couples focal scale with fisheye projection parameters, leading to ill-conditioned updates. The paper proposes CO-Calib, a framework integrating a learning-based target detector and an error-analysis-guided frame selector. CO-Calib creates initialization-friendly anchors and multi-camera constraints without altering existing calibration workflows. Experiments show CO-Calib significantly boosts success rates from 68.1% to 99.3% while improving accuracy on various systems.
