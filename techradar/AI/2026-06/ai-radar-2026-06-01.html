A new method called VLM-GLoc uses Vision-Language Models to enhance Monte Carlo Localization by leveraging rich text features, implicit quality filtering, and permanence reasoning for targeted data augmentation. VLM-GLoc achieves high global localization success rates of 70% in a grocery store and 74% in a lab space, outperforming traditional geometry-only and domain-specific baselines.

ARISTO Hand: Sensing-Driven Distal Hyperextension for Fine-Grained Manipulation

A robotic hand called ARISTO Hand is presented, featuring active distal hyperextension and a hybrid fingertip-sensing architecture. The hand's design integrates a rigid force-torque sensor with a soft capacitive tactile array. Active hyperextension increases pull-out force by 2.76x for object thicknesses of 1-20 mm without compromising grasp capability.

Any-ttach: Quick End-effector Swapping Enables Manipulation Dexterity with Simplicity

Any-ttach is a tool-centric manipulation framework that enables dexterity with simplicity by using quick end-effector swapping to facilitate task execution. The system combines an automatic swapping mechanism, a handheld device for collecting human demonstrations, and a task planning framework that composes learned skills for various tools and end-effector modules. Experiments show improved tool-swapping reliability, increased demonstration efficiency, reduced tool-pose variability, and support for diverse tool-use skills. Any-tatch executes six tool-use subskills in two long-horizon tasks through end-effector switching and execution monitoring.

Caspar: CUDA Accelerator for Symbolic Programming with Adaptive Reordering

Caspar is a library that provides a state-of-the-art nonlinear GPU solver for robotics and optimization problems, automatically generating optimized CUDA kernels from symbolic expressions. It bridges the gap between expressive Python symbolic programming and high-performance C++ GPUs. Users can define and combine symbolic expressions to generate custom CUDA kernels using Caspar's API.

ZAPS-DA: Zero-Phase Action Policy Smoothing with Decoupled Actor for Continuous Control in Reinforcement Learning

ZAPS-DA is a framework for continuous control in reinforcement learning that reduces high-frequency action jitter without introducing phase lag or post-processing. It pairs an unmodified main actor with a separate decoupled actor trained via supervised imitation of zero-phase filtered targets. The deployed policy uses causal distillation to filter actions, reducing steering jitter by 14-21x and throttle jitter by 3-5x in two driving simulators.
