Learning to Assist: Collaborative VLAs for Implicit Human-Robot Collaboration

Human-robot collaboration systems often suffer from limited scalability due to reliance on hand-engineered pipelines. This work demonstrates end-to-end imitation learning with Vision-Language-Action (VLA) models can support collaborative manipulation. Researchers identified an issue in action-chunking policies, called demonstration action leakage, which causes premature assistance during implicit HRC tasks, especially over long horizons. A proposed inference-time steering method was developed to correct these erroneous assistive actions while maintaining policy performance. User studies confirmed that steering enables longer execution and improves collaboration by preventing untimely robot interventions.

arXiv - cs.RO

Action-Effect Memory Pretraining for Robot Manipulation

Introduces AEM, an Action-Effect Memory framework for robot manipulation, designed to learn compact temporal representations from vision-action histories. Unlike methods focusing only on single frames, AEM models manipulation as an action-driven interaction by interleaving visual and action features. It applies masked modeling to reconstruct missing content from incomplete histories, learning action-conditioned state evolution. The final output is a Mamba-encoded vector serving as a global context for control decoding while maintaining efficiency. Evaluations across simulation and real-world settings show AEM consistently improves performance over existing baselines.

arXiv - cs.RO

Foresight: Iterative Reasoning About Clues that Matter for Navigation

Open-world mapless navigation from sparse language instructions requires inferring relevant environmental cues, a task limited by prior methods relying on fixed factors or missing plan-dependent clues. Foresight addresses this using a test-time framework that adapts pretrained Vision-Language Models (VLMs). This framework involves the VLM iteratively proposing image-space motion plans and then critiquing those plans based on language goals and visual context. Subsequent plans are conditioned on these critiques, allowing for refinement before physical execution. Evaluations in real-world settings showed Foresight improving average task success by 37% and reducing interventions by 52% compared to state-of-the-art baselines while maintaining real-time performance.

G-MAPP: GPU-accelerated Multi-Agent Planning and Perception for Reactive Motion Generation

Reactive motion generation in unstructured environments is computationally difficult due to collision-free planning demands. This work addresses the runtime bottleneck by proposing G-MAPP, a framework that accelerates world modeling and vector-field based planning using GPUs. The approach maintains high fidelity for perception and planning while tightly coupling the perception-action loop in real-time. Evaluation shows the GPU-based planner achieves up to a 5x speedup compared to its CPU counterpart. Furthermore, the coupled system successfully navigates complex physical scenarios without collisions using a 7-DoF robot.

arXiv - cs.RO

From Imitation to Alignment: Human-Preference Flow Policies for Long-Horizon Sidewalk Navigation

Robust long-horizon sidewalk navigation, critical for micro-mobility applications, requires precise maneuvering using minimal sensors like a single monocular RGB camera. Existing imitation learning methods struggle with compounding errors and insufficient social compliance on sidewalks. To improve this, the authors introduce FlowPilot, a mapless policy utilizing anchored flow matching trained on large-scale robot data. They further enhance it through a human-in-the-loop preference learning scheme to boost counterfactual reasoning and social adherence. Evaluations show that FlowPilot achieves strong simulated performance, and its human-tuned counterpart significantly improves real-world robustness and compliance.

arXiv - cs.RO

EgoEngine: From Egocentric Human Videos to High-Fidelity Dexterous Robot Demonstrations

EgoEngine is a novel framework designed to generate high-fidelity robotic data from scalable egocentric human manipulation videos. It addresses limitations in robot demonstration data by bridging the visual and action gaps between human actions and robotic capabilities. Given an RGB video of a person, EgoEngine synthesizes two outputs: a simulated robot observation preserving scene context, and a task-aligned, executable action trajectory respecting physical constraints. Experimental results demonstrate the framework's ability to convert raw human videos into usable robot data for learning. Notably, it achieves zero-shot visuomotor dexterous policy learning directly from egocentric human videos without needing any real-robot demonstrations.

arXiv - cs.RO

EWAM: An Enhanced World Action Model for Closed-Loop Online Adaptation in Embodied Intelligence

Introduces the Enhanced World Action Model (EWAM), a closed-loop online adaptation architecture using a frozen Cosmos3 backbone. EWAM is designed to minimize required deployment data for adapting to new task layouts via zero-shot evaluation. Its performance improvement relies on four lightweight neural layers integrated at inference time, not additional training. These modules include a Neural Experience Memory Layer, an Anomaly Detection Layer, and specialized Policy Routing and Action Correction Layers. This integration happens differentiably within the Cosmos3 forward path, with only the final routing being discrete.

arXiv - cs.RO

EquiDexFlow: Contact-Grounded SE(3)-Equivariant Dexterous Grasp Generative Flows

EquiDexFlow is an SE(3)-equivariant flow-matching model designed to jointly generate kinematically-plausible dexterous grasps while respecting physical constraints. It predicts wrist pose, joint angles, fingertip contacts, surface normals, and contact forces directly from an object point cloud. The architecture inherently ensures that grasp placements comply with the object surface and predicted forces respect the Coulomb friction cone. Trained on 8,100 force-closure grasps for the Allegro Hand, it achieves zero friction violations and low wrench residuals. When retargeted to a LEAP Hand and tested physically, EquiDexFlow successfully completes open-loop pick-and-hold trials across six different objects, including asymmetric ones.

arXiv - cs.RO

Sparse2Act: Learning Action-Aligned Sparse 3D Representations for Cross-Domain Robot Manipulation

Sparse2Act is a framework that pretrains sparse point-cloud encoders using task-space end-effector actions as geometric supervision. It trains masked sparse 3D tokens to organize scene features around observed workspace motions and paired observations. After pretraining, only the encoder initialization is reused by downstream policies, preserving their original architectures and action spaces. On LIBERO-10, the method achieved an average success rate of 86.9% after fine-tuning. Furthermore, it demonstrated strong cross-domain transfer, reaching 73.4% on Meta-World-5 from the LIBERO dataset, and showed 72.5% success in real-world tasks.
