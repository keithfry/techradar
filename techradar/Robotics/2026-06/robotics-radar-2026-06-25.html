ROS Discourse General: VL53L0X Time-of-Flight sensor as a Nav2 obstacle layer (ESP32 + ROS 2 Humble)

Details an open-source project integrating the low-cost VL53L0X Time-of-Flight sensor into the Nav2 costmap as an obstacle source for small robots. The system pipeline connects the sensor to an ESP32, streams data via serial to ROS 2, converts it to a `LaserScan` message, and feeds it into the `ObstacleLayer`. Included components are custom firmware, a serial bridge node, and a converter that synthesizes a fan from the single beam. The implementation was tested on ROS 2 Humble and provides configuration advice for setup pitfalls.

arXiv - cs.RO

Invariant Kalman filtering for extended pose estimation in multi-IMU articulated rigid-body systems

Accurate extended pose estimation for multi-IMU articulated systems is challenging due to inter-body coupling and incorporating joint constraints. The authors introduce the relative L-extended pose, a Lie group representation allowing joint constraints to be expressed in an invariant form. This framework is utilized within an iterated Invariant Extended Kalman Filter (IterIEKF). IterIEKF incorporates these invariant constraints as noise-free pseudo-measurements while maintaining filtering consistency guarantees. Testing on a UR5e robot and human leg demonstrates that IterIEKF outperforms existing EKF and IterEKF baselines, achieving significantly lower RMSE and faster convergence.

ADM-Fusion: Adaptive Deep Multi-Sensor Fusion for Robust Ego-Motion Estimation in Diverse Conditions

Developing reliable autonomy in changing environments requires adaptive multi-sensor fusion for ego-motion estimation. The proposed ADM-Fusion is an end-to-end deep learning method designed to handle environmental changes and sensor failures. It uses an adaptive sensor mixture-of-experts framework with content-aware routing to dynamically weigh different sensor inputs. Furthermore, the system includes separate translation and rotation branches connected by a cross-task attention mechanism for specialized information sharing. Testing on CARLA-LOC and KITTI data confirms ADM-Fusion's robust performance despite degraded operating conditions.

arXiv - cs.RO

AeroCast: Probabilistic 3D Trajectory Prediction for Non-Cooperative Aerial Obstacles via Transformer-MDN Architecture

Autonomous aerial vehicles require probabilistic methods to predict the unpredictable movements of non-cooperative obstacles like birds or debris. AeroCast introduces a framework combining a Transformer encoder with a Mixture Density Network (MDN) output head for this purpose. It predicts per-timestep Gaussian mixture distributions representing future 3D displacements, moving beyond single-point estimates. The model addresses input design and mode-degeneracy using consecutive displacement encoding and a calibration-oriented loss function. Tested on quadrotor data, AeroCast substantially reduces prediction errors while maintaining real-time inference speeds compatible with high-frequency onboard deployment.

arXiv - cs.RO

RGB: RL Guided Whole-Body MPPI for Humanoid Control

Develops an RL guided Whole-Body Model Predictive Path Integral (MPPI) framework for humanoid control. The method utilizes a pre-trained Reinforcement Learning (RL) policy as a sampling prior to bias MPPI rollouts towards feasible behaviors, rather than using it as the final controller. Task objectives are integrated via modular MPPI cost terms, allowing online correction of the RL prior without retraining. Simulations on a 29-DoF humanoid show stable, high-rate control (averaging 280~Hz) in MuJoCo. This approach improves task precision by correcting systematic drift and tracking additional whole-body reference signals over pure RL methods.

Causality-Based Parametric Control Barrier Function for Safe Multi-Vehicle Interaction

Addresses challenges in safe control for autonomous driving involving multiple vehicles and varying behaviors. Existing methods struggle to model inter-vehicle influence causality or suffer from overly conservative worst-case analysis. This research extends Parametric-CBF by incorporating embedded causality inference to explicitly reason about interactions between surrounding vehicles. A resulting adaptive safety controller uses this causality-based function to allow the ego vehicle safe reactions based on learned neighbor expectations. Testing shows that utilizing motion flexibility significantly improves task efficiency in complex multi-vehicle scenarios.

Memory Retrieval in Visuomotor Policies for Long-Horizon Robot Control

Autonomous robots in partially observable environments need general memory retrieval mechanisms to recall diverse past information for long-term control. Incorporating transformer-based memory into imitation learning faces challenges, including learning spurious correlations and compounding memory errors leading to model drift. To address this, the proposed method, HALO, integrates an attention-based memory system for visuomotor policies. It suppresses false correlations by distilling vision-language model priors using video question-answering objectives on demonstration data. Furthermore, HALO mitigates accumulated error by employing sparse attention, allowing reliable retrieval from up to eight minutes of past experience.

Toward Low-Latency Vision-Language Models with Doubly-Correct Predictions in Egocentric Visual Understanding

Low-latency inference in Vision-Language Models (VLMs) is crucial for real-time human-robot collaborative tasks. Weight pruning techniques can improve VLM efficiency for on-board processing while ensuring safe human-robot interaction. Safe interaction requires preserving "doubly-correct predictions"—outputs that are both accurate and evidentially grounded. The paper introduces a rationale-informed pruning strategy because existing methods often lose prediction accuracy despite retaining evidence localization. Benchmarks confirm this new method achieves superior prediction accuracy and doubly-correct performance on egocentric video datasets.
