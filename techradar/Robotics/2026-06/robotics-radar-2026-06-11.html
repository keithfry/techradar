ROS Discourse General: EpisodeVault: open source tool to debug why your LeRobot model regressed

Introduces EpisodeVault, an open-source tool designed to debug model regressions in LeRobot by analyzing dataset changes. It allows users to track datasets and commit versions, facilitating comparison between different states via the `diff` command. The difference report specifically highlights additions/removals of episodes, shifts in task distribution, and changes in quality metrics (e.g., success rate). A dedicated `blame` command traces a specific model back to the precise dataset version it was trained on. Installed via pip, EpisodeVault supports various local LeRobot v3 datasets.

MASK: Multi-Agent Semantic K-Scheduling for Risk-Sensitive 6G Robotics

To achieve collaborative control in 6G robotics despite limited wireless spectrum, this work introduces Multi-Agent Semantic K-Scheduling (MASK). MASK utilizes Arbiter-Assisted Semantic Information Gating (A-SIG) to schedule only the top-K agents based on their semantic importance scores under strict bandwidth caps. These prioritized observations are then condensed into a latent state for a self-supervised global encoder. This setup enables a distributional policy to mitigate risks even with data sparsity. Evaluations confirm MASK matches unconstrained baselines even when resource access is heavily restricted, proving its utility in resource-limited 6G systems.

arXiv - cs.RO

Model-based Optimization of Anguilliform Swimming Gaits for Soft Robotic Applications

Introduces the Soft Lamprey-Inspired Dual Environment Robot (SLIDER) along with its modeling and optimization design procedures. The method uses Lighthill's theory to model fluid environment actions like inertial effects and vortex forces, while developing a nonlinear model for structural parameters. Fluid-structure interaction equations are solved implicitly using an efficient second-order box method for pneumatic actuation in water. Testing revealed that low-frequency swimming is dominated by environmental resistance, while high-frequency motion is primarily influenced by inertial fluid forces. Co-optimization of control patterns and caudal fin design achieved a tethered swimming speed of 21.7 +/- 0.4 cm/s, with investigation also covering multimodal tasks.

Embodied-R1.5: Evolving Physical Intelligence via Embodied Foundation Models

Introduces Embodied-R1.5, a unified Embodied Foundation Model (EFM) designed for general physical intelligence, integrating reasoning across cognition, planning, correction, and pointing. The model utilized over 15B tokens from three automated data construction pipelines and employs a multi-task balanced RL recipe to manage task conflicts. It features a Planner-Grounder-Corrector (PGC) closed-loop framework enabling autonomous execution and self-correction on long tasks. Despite having only 8B parameters, Embodied-R1.5 achieved state-of-the-art results on 16 of 24 embodied VLM benchmarks. Furthermore, it shows strong generalization in zero-shot real-robot experiments and has open-sourced its weights and a specialized evaluation kit.

HiPi: Reproducible High-Fidelity Piezoresistive Sensors for Robotic Manipulation

Introduces HiPi, a new system designed to bridge the gap between reproducible and high-fidelity piezoresistive tactile sensing for robotics. HiPi addresses prior trade-offs by emphasizing deployability through a compact, commercial PCB readout module and an STM32-based MCU. The system features FPCB-based conductive layers simplifying sensor fabrication and stacking. Benchmarking demonstrated that using structured 3D-printed patterns with HiPi significantly improved contact geometry preservation compared to a reproducible baseline. These results suggest HiPi makes dense piezoresistive tactile sensing more practical for complex, multi-fingered robotic manipulation.

arXiv - cs.RO

PLUME: Probabilistic Latent Unified World Modeling and Parameter Estimation for Multi-Finger Manipulation

Dexterous multi-finger manipulation is highly sensitive to unknown physical parameters like object shape or friction, posing a challenge for deploying simulation-trained policies. Proposes PLUME, which is a world model that jointly learns the system dynamics and maintains a belief over uncertain parameter values. It uses a latent space to represent multiple physical parameters and rewards derived from partially observable variables. This framework enables efficient alignment of the world model to true dynamics via online parameter inference without retraining. Evaluations across tasks like screwdriver turning demonstrate successful zero-shot transfer on hardware, outperforming existing state-of-the-art methods.

arXiv - cs.RO

Dynamic Execution Horizon Prediction for Chunk-based Robot Policies

Action chunking, common in modern robot policies, assumes a fixed action execution horizon during open-loop operation. This assumption is problematic for complex tasks requiring frequent replanning, as the horizon is typically tuned empirically. The proposed Dynamic Execution Horizon Prediction (DEHP) method addresses this by training a lightweight prediction branch using online reinforcement learning while freezing the existing chunk policy. DEHP remains compatible with black-box policies and isolates horizon adaptation from action generation changes. Evaluations show DEHP significantly boosts success rates across high-precision, long-horizon tasks by dynamically adjusting the predicted horizon based on task needs.

A Modular Dual-Camera Pipeline for Micro-Inspection Using Aerial Robots

Addresses limitations of current drone inspection systems that require close proximity or detailed prior knowledge of targets. Introduces a modular pipeline, *aerial\_micro\_inspection*, designed for diverse micro-inspection tasks. The system utilizes two cameras: a zoomed gimbal camera for fine details and a wide-view stereo camera for surface mapping and regional partitioning. A vision-based feedback loop maintains stability while the inspection camera surveys these smaller regions. Evaluation in simulations and real tests confirmed its robustness, successfully detecting caterpillars on trees and whiteflies in greenhouses.

Bridging the sim2real gap in the table tennis robot with a transformer-based ball states predictor

Predicting ball states accurately is crucial for controlling robotic table tennis in dynamic settings. Previous physics-based methods require precise parameters, and learning models struggle with long temporal dependencies. The presented work introduces a transformer framework using attention mechanisms to model these correlations from historical data without explicit flight modeling. To ensure robustness, the authors utilized a large-scale dataset collected from real human players. A novel strategy called Swap Predictor at Deployment (SPAD) successfully minimizes the sim-to-real gap by replacing the training simulator with the real-world predictor during deployment.

arXiv - cs.RO

Steering Multirobot Behavior via Closed-Loop Affine Activation Editing

Closed-Loop Affine Activation Editing (CLAE) is an inference-time framework designed to modify the behavior of frozen, pre-trained robotic policies without retraining or risking catastrophic forgetting. CLAE treats behavior steering as a closed-loop problem, adjusting intermediate activations based on real-time robot state and environment information. It achieves this by training a sparse autoencoder over policy activations and learning a lightweight reinforcement learning (RL)-based policy to apply state-dependent affine edits. Validation using a multi-quadrotor navigation task demonstrated CLAE's ability to control individual robot velocity profiles, maintain desired formations for coordinated movement, and generate novel behaviors like reducing camera exposure.

arXiv - cs.RO

Learning Object Manipulation from Scratch via Contrastive Interaction

Contrastive Reinforcement Learning (CRL) struggles in complex manipulation due to object interactions causing distinct changes in dynamic modes. The paper models manipulation dynamics as a piecewise-smooth Markov process, noting that mode changes create difficult piecewise nonlinear reachability structures. To address this, Interaction-weighted Resampling (IWR) is introduced, which performs interaction-aware resampling around phase transitions. IWR encourages learned representations to preserve these critical mode boundaries for multi-modal planning. Evaluations show IWR improves sample efficiency and performance across various interactive environments, achieving a 60% success rate in real-world robot air hockey tasks via sim-to-real transfer.
