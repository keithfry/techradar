Introduces a novel random feature construction for Bernstein-Schur kernels, which are products of finite-feature and completely monotone shift-invariant kernels. The method achieves this by simultaneously sketching the finite modulation factor and applying Gaussian random Fourier features to the one-dimensional Bernstein-Widder scale of the radial factor. PermDoRA -- Understanding Adapter Interference in Language Models: Limits of Parameter-Space Geometry

Effective access control in large language models necessitates modular methods to prevent cross-domain interference without retraining. Researchers tested the hypothesis that linear overlap causes adaptation interference using a framework called DoRA-RBAC on LLaMA-3.1-8B and Mistral-7B across various QA benchmarks. Comparing standard Euclidean merging with geometry-aware, Riemannian-inspired averaging showed no consistent performance benefit for the latter in multi-domain settings. The study concludes that adapter interference is likely due to interactions within shared nonlinear representations rather than solely being governed by parameter-space geometry like orthogonality.

Annealed Entropic Allocation for Ranking and Selection

Introduces Annealed Entropic Allocation, a weighted soft-min framework for sequential budget allocation in ranking and selection. It replaces the non-smooth maximin large-deviation rate objective with a weighted log-sum-exp surrogate that handles near-active challengers better. The method incorporates a saddlepoint approximation to enhance finite-budget discrimination using sub-exponential corrections. This surrogate maintains the same first-order target as the classical maximin formulation because the smoothing parameter is annealed to zero. Experiments show competitive performance, particularly when multiple challenger scores are closely grouped.

Enhancing Spectral Embedding through Robust and Flexible Knowledge Transfer in Electronic Health Records

Presents a spectral-based, unsupervised learning framework for creating low-dimensional embeddings of clinical concepts and patients from limited electronic health records for rare diseases. To address data scarcity, it integrates a knowledge matrix derived from a larger population with partial overlap to the target cohort. Uniquely, the method abandons strict one-to-one signal alignment, permitting flexible structural knowledge sharing. It employs a novel two-step spectral embedding process: first cleaning irrelevant components from the knowledge matrix, then projecting to separate shared and heterogeneous information. Testing on multiple sclerosis data demonstrates superior performance compared to existing methods, especially when shared signals are weak or partially aligned.

Renewable Lasso without Batch-Number Constraints: A Gradient-Enhanced Approach

Proposes a gradient-enhanced surrogate loss for online estimation of high-dimensional generalized linear models using streaming data. For non-distributed settings, this method improves upon prior renewable estimation techniques by removing the batch-number constraint. It extends to distributed streaming data in a master-client architecture, exchanging only summary gradient vectors. The adjusted approach avoids requiring clients to compute the full surrogate loss. Derived non-asymptotic error bounds validate the method's performance without strict limitations on batch count, showing improved accuracy over existing estimators.

Conformal Bayes under Label Shift: Post-Hoc Calibration vs. In-Training Adaptation

Conformal Bayes integrates Bayesian posteriors with conformal calibration to generate statistically valid and efficient prediction sets. When addressing label shift, the work analyzes two methods: post-hoc calibration and in-training adaptation. Post-hoc calibration corrects the conformal threshold using importance weights while keeping the parameter posterior fixed. In contrast, in-training adaptation adjusts the entire parameter posterior towards the target domain. Experiments demonstrate that both strategies yield valid coverage under unbiased training, but in a lead-optimization scenario, in-training adaptation debiases the model by narrowing interval width without losing coverage.

From Persistence to Survival: Hypothesis Testing, Effect Sizes and Vectorisation for Topological Features

Persistence diagrams, key tools in topological data analysis, lack natural vector space representations for comparative statistics. A novel approach called STRAND treats collections of these diagrams as survival data, using the persistence survival function $S(t) = \mathbb{P}(p > t)$ as its core metric. From this single representation, the method derives a non-parametric two-sample test and interpretable effect sizes for hypothesis testing. Furthermore, it generates a 1-Wasserstein-stable feature vector suitable for machine learning applications. Validation across synthetic data and benchmarks in graph and 3D point clouds demonstrates STRAND as a unified framework for topological analysis.

Phase Transitions in Attention: A Bayesian Theory of Copy Head Emergence

Attention, central to in-context learning in transformers, often shows patterns emerging abruptly during training. Researchers present a Bayesian theory of feature learning within attention, specifically examining how a copy subcircuit emerges in an induction head's first layer. By deriving a closed-form posterior for the attention matrix, they reduce the problem to a low-dimensional parameter space. This reveals a phase transition in the required amount of training data, verified through both Bayesian sampling and standard Adam training. The findings contrast softmax attention exhibiting a first-order phase transition with linear attention's initial second-order transition followed by smooth evolution.

Fixed-Parameter Tractability of Private Synthetic Data Generation

Fixed-parameter tractability (FPT) is established for generating synthetic data while satisfying differential privacy constraints. The specific parameter driving the tractability is the treewidth of the query family's incidence graph. Two distinct methods achieve optimal error rates: one utilizes linear programming and the FPT of the associated LP dual, and the other employs a subsampled private multiplicative weights method for Gibbs distribution sampling. Both algorithmic approaches are unified through a dynamic programming framework built over a tree decomposition.

Quantized Stochastic Primal-Dual Methods for Distributed Optimization under Relaxed Global Geometry

Presents a quantized stochastic primal-dual method, q-PDGD, for distributed optimization involving stochastic gradients and finite-bit communication via random quantization. Analyzes this method under relaxed global geometry conditions. Under restricted secant inequality, constant step sizes guarantee linear contraction to a neighborhood dependent on noise and quantization error. With diminishing step sizes, the method achieves $O(1/k)$ convergence even without requiring shared minimizers. The proposed rates match state-of-the-art centralized stochastic rates in terms of oracle complexity.

GraphGP: Scalable Gaussian Processes with Vecchia's Approximation

Gaussian processes are powerful but computationally costly due to their $\mathcal{O}(N^3)$ time complexity. GraphGP addresses this by implementing Vecchia's approximation, which conditions each point only on its $k$ nearest neighbors using a sparse precision matrix. This GPU algorithm achieves linear time and memory scaling, enabling use with very large datasets. Key technical advancements include a bit-reversed k-d tree ordering for efficient neighbor searches and maximizing batch parallelism. The method provides necessary building blocks for inference tasks like forward generation and log-determinant calculation via a differentiable CUDA implementation.

Signed Compression Progress on a Sealed Audit is Goodhart-Resistant

Signed compression progress serves as a theoretically sound intrinsic motivation reward for agents improving their world model predictions. If this reward is defined by the signed decrease in a fixed, sealed-audit loss ($\text{r}_t = E(\theta_{t-1}) - E(\theta_t)$), cumulative rewards precisely track true audit improvement. For finite panels, empirical reward is bounded by true audit improvement plus a term related to model class deviation. This method proves robust against various failure modes, including clipping or scoring on the agent's own data stream. Overall, signed compression progress provides an accounting signal guaranteeing genuine performance enhancement when audited by a sealed panel.
